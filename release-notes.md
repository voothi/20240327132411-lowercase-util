
## v1.0.0 (2026-01-02)
**Initial Release**

- **Feature**: **Lowercase Conversion**. Converts input text or clipboard content to lowercase.
- **Feature**: **HTML Tag Removal**. Automatically strips HTML tags (`<br>`, `<div>`, etc.) from the input.
- **Feature**: **Control Character Removal**. Removes non-printable characters and control codes.
- **Feature**: **Newline Normalization**. Replaces single newlines with spaces to flatten text, while preserving paragraph breaks (newlines followed by whitespace).
- **Feature**: **Clipboard Integration**. Reads from and writes back to the system clipboard for a seamless workflow.
- **Feature**: **Command Line Interface**. Supports direct input via command line arguments or interactive use with the clipboard.
